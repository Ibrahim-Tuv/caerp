Automatic Leave Allocation for employee

This module provides configuration for allocation of leaves and automated job to allolcate leave based that configuration.

Automated job for leave allocation.

Configurable leave allocation for different leave types.

Configure Leave allocation > Time off > Automatic Leave Allocation
  • Leave Type: Select leave type from this dropdown.
  • Type: Choose Type for allocation from Full or Pro-rata allocation.
  • By Employee: Select one or more employees from employee tab for leave allocation.
  • By Company: Leave allocation based on selected Company.
  • By Department: Option to allocate leaves department wise, choose department from tab department.
  • By Tags: Allocate leaves based on employee tag, choose tags from tab.
  • No of Days / No of Hours: Enter number of days/hours to allocate.






Full Leave allocation By employee

Using Full type of allocation can allocate leaves for employee for full year.

Pro-rata allocation By employee

Using Pro-rata type of allocation can allocate leaves monthly.
